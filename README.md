# coffee-club
Hackspace - The Coffee Club

# Setup

1. Clone it

	git clone git@github.com:brenopolanski/coffee-club.git
	
2. Install Dependencies
	
	cd coffee-club && npm install

3. Serve it

	$ node server.js
